Сильные глаголы (starke Verben) изменяют корневую гласную при образовании основных форм:

|               |             |                 |
| ------------- | ----------- | --------------- |
| g<u>e</u>hen  | g<u>i</u>ng | geg<u>a</u>ngen |
| s<u>i</u>tzen | s<u>a</u>ß  | ges<u>e</u>ssen |
и спряжении:

|     |              |
| --- | ------------ |
| ich | g<u>e</u>be  |
| du  | g<u>i</u>bst |
| er  | g<u>i</u>bt  |
